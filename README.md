Decentralized IoT Solution for Smart Agriculture: 
A decentralized smart irrigation system using Raspberry Pi and
ESP32 nodes, equipped with soil moisture sensors and sprinklers. 
The system operates on a master-slave architecture, using a local 
network for communication. A private web server enables real-time 
monitoring and automated moisture-based sprinkler control, 
ensuring data privacy, reducing water waste, and eliminating 
third-party cloud dependency for sustainable farming.
